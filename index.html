Basic Lines

These are the basic lines. There is a mixin associated with these lines, it takes in the color you'd like the lines to be and also the line-height of the element. They use the before pseudo element so that the after pseudo element is avaiable for other things. See the Other Goodies section.

Mixins to use

For Dotted: @include dotted($dot-color, $size);
For Crossed: @include cross($color, $size);
For Connected: @include connected($line-color, $dot-color);
! These should all use an RGBA value.
